# vue + element 响应式后台模板

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```



## 主要功能

- 响应式侧边栏
- 面包屑导航（结合router.js）
- 路由动效
- 阿里巴巴矢量icon引入
- 公用样式
- stylus预处理CSS
- 路由懒加载
- 路由拦截
- axios请求拦截
- vuex 状态分模块管理
